HeritrixRemote

Java application to control your Heritrix

Download this project as a .zip file Download this project as a tar.gz file

Currently under development! :-)

1. Introduction

HeritrixRemote is a command line application written in Java, which can help you control your running Heritrix crawler, especially when you've got a lot of crawling jobs to manage.

2. Requirements

  • installed JDK 1.6 or newer - to run this stuff
  • installed cURL - need to communicate with Heritrix
  • running Heritrix 3.x - you will control this
